## 推荐阅读顺序

**如果你是第一次接触本项目**，建议按以下顺序阅读：

1. **[ARCHITECTURE.md](./ARCHITECTURE.md)** — 先建立全局视角，了解组件如何协作、数据如何流转
2. **[智能体模型运作流程与框架.md](./智能体模型运作流程与框架.md)** — 深入理解模型路由与生成流程
3. **[模型选型建议.md](./模型选型建议.md)** — 了解各角色的模型选择逻辑与成本考量
4. **[测试与验收.md](./测试与验收.md)** — 跑一遍离线冒烟测试，验证本地环境可用

**如果你只关心部署**：直接看 [mysql-storage-setup.md](./mysql-storage-setup.md)。

**如果你只关心模型调优**：从 [模型选型建议.md](./模型选型建议.md) 和 [模型评分榜.md](./模型评分榜.md) 开始。
